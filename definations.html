What is calc() and why it is used
calc() ek CSS function hai jo mathematical calculations perform karne ke liye use hota hai. Iska use responsive layouts banane ke liye hota hai jahan fixed aur flexible units ko mix karna zaroori ho.
Using calc() with width and height
calc() width aur height ke sath use hota hai taake elements ka size dynamically adjust ho sake, jaise percentage aur pixel ko combine karna.
100vh - header height
Ye formula tab use hota hai jab header fixed ho aur baqi page ka content screen ki remaining height occupy kare.
Creating a layout
Layout webpage ka structure hota hai jisme header, sidebar, content aur footer arrange kiye jate hain.
Converting the layout
Layout conversion ka matlab hota hai existing layout ko kisi aur system jaise Flexbox se Grid me change karna.
Implementing layout using CSS
Layout implementation CSS ke zariye hoti hai jisme Flexbox ya Grid ka use karke elements ko arrange kiya jata hai.
Difference between Flex and Grid
Flexbox one-dimensional layout system hai jo row ya column me kaam karta hai, jab ke Grid two-dimensional system hai jo rows aur columns dono ko control karta hai.
What are keyframes
@keyframes CSS animations ke steps define karne ke liye use hote hain jo animation ke start aur end points batate hain.
Creating animations using @keyframes
@keyframes ke zariye animations create ki jati hain jisme element dheere dheere ek state se doosri state me jata hai.
Applying animations (rotate, scale)
Animations apply karne ke liye transform properties jaise rotate aur scale use ki jati hain jo element ko move ya resize karti hain.
What are media queries and why they are important
Media queries CSS rules hain jo screen size ke mutabiq styles apply karti hain, jo responsive design ke liye bohat zaroori hoti hain.
Media queries for mobile, tablet, and desktop
Mobile, tablet aur desktop ke liye different breakpoints define karke CSS likhi jati hai taake website har device par properly display ho.
